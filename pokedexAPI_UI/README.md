# PokeAPI Next.js Pokedex

This is a Next.js Pokedex activity that uses the PokeAPI.

This folder contains the Pokedex API integration and UI.

## Features

- Search Pokemon by name or ID
- Display Pokemon image
- Display Pokemon height
- Display Pokemon weight
- Display Pokemon type and ID
- Previous and next Pokemon navigation

## Pokedex Files

The Pokedex page is located at:

```txt
app/pokedex
```

Main files:

```txt
app/pokedex/page.tsx
app/pokedex/PokedexClient.js
app/pokedex/pokedex.css
```

## How to Run

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open this page in the browser:

```txt
http://localhost:3000/pokedex
```

## API Used

```txt
https://pokeapi.co/
```
