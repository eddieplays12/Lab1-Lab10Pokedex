## PokeAPI Next.js Pokedex

The Pokedex frontend with PokeAPI integration is located inside:

```txt
pokedexAPI_UI
```

The Pokedex page files are located inside:

```txt
pokedexAPI_UI/app/pokedex
```

## How to Run the Pokedex

Open a terminal and go to the Next.js frontend folder:

```bash
cd pokedexAPI_UI
npm install
npm run dev
```

Then open this page in the browser:

```txt
http://localhost:3000/pokedex
```

## API Used

This activity uses PokeAPI:

```txt
https://pokeapi.co/
```

The Pokedex displays the Pokemon image, height, weight, type, and ID.
